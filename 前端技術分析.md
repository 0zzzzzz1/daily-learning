| 2026-04-27 | 箭頭函式 `this` | 「this 很難搞」 | 理解靜態綁定機制，知道何時不能用箭頭函式（Vue options API 方法）| ⬆⬆ 明顯 |

| 2026-05-03 | Vue 3 Lifecycle Hooks | `onMounted` 就是「元件出現時執行」 | 理解完整生命週期時間軸；SSR 中 `onMounted` 不執行；`watchEffect` 早於 `onMounted` | ⬆⬆⬆ 顯著 |
| 2026-05-03 | `watchEffect` 依賴追蹤 | 以為整個 async callback 都能追蹤依賴 | 依賴追蹤只在第一個 `await` 之前的同步程式碼有效 | ⬆⬆ 明顯 |
| 2026-05-03 | Composable 中的生命週期 | 「Lifecycle 只能在元件裡用」 | Vue 的 Active Instance 機制讓 Composable 可以封裝 `onMounted`/`onUnmounted` | ⬆⬆⬆ 顯著 |
| 2026-05-04 | W2 整週知識整合 | 各知識點分散、彼此獨立 | 非同步（W2 理論）→ 實作（async-utils）→ Vue 整合 → 生命週期 → Composable 封裝：一條完整的知識鏈 | ⬆ 整合提升 |
